# IPL Analytics Dashboard

A comprehensive data analytics dashboard built with Python and Streamlit, analyzing 16 years of IPL cricket (2008-2024).

## Features
- 6 interactive tabs covering overview, team analysis, H2H, venues, prediction, and era comparison
- Uses both matches.csv and deliveries.csv (260,000+ ball-by-ball records)
- ML-powered match predictor with engineered features

## Tech Stack
Python | Pandas | Scikit-learn | Matplotlib | Seaborn | Streamlit
